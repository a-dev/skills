This repository contains Agent Skills plus small executable harness artifacts. `package.json` runs the read-only audit tests with Node's built-in test runner; there is no application build.

`skills/frontend/css-modules/SKILL.md` - per-edit CSS Modules conventions; model-invocable while styling.
`skills/frontend/css-modules-setup/SKILL.md` - manual one-time bootstrap + alignment audit (`/css-modules-setup`, `disable-model-invocation: true`).

The canonical project-profile schema, templates, audit script, and fixtures live under `skills/frontend/css-modules-setup/`. Keep executable claims in the Markdown synchronized with those artifacts.
